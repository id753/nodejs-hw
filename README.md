## Створено Express-додаток для роботи з колекцією нотаток і реалізовано необхідні HTTP-обробники.

Добре реалізовані аспекти:

    Коректне використання ES-модулів та імпортів.
    Змінні середовища завантажуються за допомогою 'dotenv/config'.
    Сервер слухає на правильному порту, за замовчуванням — 3000, якщо PORT не встановлено.
    CORS увімкнено для всіх маршрутів.
    Додано middleware express.json().
    Логування HTTP-запитів інтегровано за допомогою pino-http з чіткою конфігурацією.
    Усі необхідні маршрути (/notes, /notes/:noteId, /test-error) реалізовано з відповідною структурою JSON-відповідей.
    Middleware для обробки невідомих маршрутів повертає статус 404 та JSON-повідомлення.
    Middleware для обробки помилок повертає статус 500 і JSON-відповідь.
    Повідомлення про запуск сервера показує порт.
Задеплоєно проєкт на render.com
